## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

依赖包包括：
- PyQt5 >= 5.15.0 (图形界面)
- pynput >= 1.7.6 (鼠标键盘控制)
- pyautogui >= 0.9.53 (自动化操作)
- json5 >= 0.9.0 (JSON 处理)

### 2. 启动程序

**方法 1: 使用启动脚本**
```bash
启动.bat
```

**方法 2: 命令行启动**
```bash
python main.py
```

### 3. 使用功能

1. **鼠标连点**: 选择"🖱️ 鼠标连点"选项卡 → 设置参数 → 点击"🌸 开始连点"
2. **键盘连点**: 选择"⌨️ 键盘连点"选项卡 → 设置参数 → 点击"🌸 开始连点"
3. **录制操作**: 选择"📹 录制操作"选项卡 → 点击"🌸 开始录制" → 执行操作 → 停止录制
4. **回放操作**: 选择"▶️ 回放操作"选项卡 → 加载录制文件 → 点击"🌸 开始回放"

## 📖 使用文档

### 鼠标连点使用

1. 打开"🖱️ 鼠标连点"选项卡
2. 选择鼠标按钮（左键/右键/中键）
3. 设置点击间隔（毫秒）
4. 设置点击次数（0 表示无限循环）
5. 点击"🌸 开始连点"按钮
6. 按 **F6** 可快速启动/停止

### 键盘连点使用

1. 打开"⌨️ 键盘连点"选项卡
2. 选择要连点的按键
3. 可选择修饰键（Ctrl/Shift/Alt）
4. 设置点击间隔和次数
5. 点击"🌸 开始连点"按钮
6. 按 **F7** 可快速启动/停止

### 录制操作使用

1. 打开"📹 录制操作"选项卡
2. 选择要录制的操作类型
3. 点击"🌸 开始录制"按钮（或按 **F9**）
4. 执行需要录制的操作
5. 点击"🌸 停止录制"按钮（或按 **F10** / **F9**）
6. 如有误操作，按 **Ctrl+Z** 撤销
7. 保存录制文件（**Ctrl+S**）

### 回放操作使用

1. 打开"▶️ 回放操作"选项卡
2. 加载已保存的录制文件（**Ctrl+O**）
3. 设置回放速度（1%-500%）
4. 设置循环次数（1-1000 次）
5. 点击"🌸 开始回放"按钮（或按 **F11**）
6. 按 **F12** 或 **F11** 可停止回放

### 脚本编辑器使用

1. 按 **Ctrl+E** 或菜单"文件" → "脚本编辑器"
2. 加载录制文件
3. 在表格中查看和编辑动作
4. 可添加、删除、修改动作
5. 保存修改后的文件

## 🔧 高级功能

### 撤销/重做功能 ⭐

**录制操作错误可以重来！**

- **撤销**: 按 **Ctrl+Z** 撤销最后录制的动作
- **重做**: 按 **Ctrl+Y** 重做撤销的动作
- **批量删除**: 点击"🗑️ 删除最后 10 个动作"按钮
- **状态显示**: 实时显示"已录制：X | 可撤销：Y | 可重做：Z"

**使用场景**:
```
1. F9 - 开始录制
2. 执行操作（误操作了）
3. F10 - 立即停止
4. Ctrl+Z - 撤销误操作
5. Ctrl+Y - 如撤销过多可重做
6. Ctrl+S - 保存
```

### 全局快捷键 ⭐

**程序启动后，在任意界面都可以使用快捷键！**

无需切换到对应选项卡，直接按快捷键即可：
- 在"鼠标连点"页面按 F9 → 直接开始录制
- 在"键盘连点"页面按 F11 → 直接开始回放
- 在任何页面按 Esc → 紧急停止所有操作

## 📊 文件格式

### 录制文件格式

录制文件使用 JSON 格式，示例：

```json
{
  "metadata": {
    "created_at": "2026-03-25T10:30:00",
    "action_count": 10,
    "duration": 5.5
  },
  "actions": [
    {
      "type": "mouse_move",
      "x": 500,
      "y": 300,
      "time": 0.0
    },
    {
      "type": "mouse_click",
      "x": 500,
      "y": 300,
      "button": "Button.left",
      "pressed": true,
      "time": 0.5
    },
    {
      "type": "key_press",
      "key": "Key.a",
      "time": 1.0
    }
  ]
}
```

**动作类型**:
- `mouse_move` - 鼠标移动
- `mouse_click` - 鼠标点击
- `mouse_scroll` - 鼠标滚动
- `key_press` - 键盘按下
- `key_release` - 键盘释放

## 🐛 故障排除

### 问题：程序无法启动
- 检查 Python 版本是否 >= 3.6
- 确认已安装所有依赖包：`pip install -r requirements.txt`
- 查看错误信息

### 问题：连点器不工作
- 检查是否以管理员身份运行
- 确认目标窗口不是管理员级程序
- 尝试调整点击间隔

### 问题：录制/回放不准确
- 确保屏幕分辨率一致
- 检查是否有其他程序干扰
- 尝试调整回放速度

### 问题：快捷键不起作用
- 确保程序窗口处于激活状态
- 检查是否在其他全屏应用后面
- 确认快捷键未被其他程序占用
