鼠大师 - 使用说明

软件功能：
====================
- 定点连点：在指定坐标执行点击操作，支持脚本回放
- 脚本录制：录制并回放鼠标/键盘操作
- 脚本管理：管理和编辑录制的脚本

使用方法：
====================
1. 启动软件：
   - 直接运行 dist/鼠大师.exe 文件
   - 或者使用源代码运行：python main.py

2. 主要功能：
   - 定点连点：点击"开始添加定点"按钮，然后使用 Alt 键或 Ctrl 键添加坐标
   - 录制脚本：点击"开始录制"按钮
   - 回放脚本：点击"开始回放"按钮

3. 快捷键：
   - F9：开始/停止录制
   - F10：停止录制
   - F11：开始/停止回放
   - F12：停止回放
   - Ctrl+S：保存录制文件
   - Ctrl+O：加载录制文件
   - Ctrl+E：打开脚本编辑器
   - Ctrl+Z：撤销录制
   - Ctrl+Y：重做录制
   - F1：暂停/继续定点连点
   - F2：全局紧急停止
   - Esc：停止所有操作
   - F8：停止所有操作

4. 定点功能说明：
   - Alt 键：添加纯左键点击的坐标点（不加载脚本）
   - Ctrl 键：添加需要加载脚本的坐标点

注意事项：
====================
- 软件需要管理员权限才能正常使用
- 在 Windows Defender 或其他杀毒软件中可能会被误报，请允许访问
- 如果软件无法运行，请尝试以管理员身份运行

文件说明：
====================
- dist/鼠大师.exe：主程序可执行文件
- scripts/：脚本文件夹（包含示例脚本）
- main.py：源代码
- 使用说明.txt：本文件
